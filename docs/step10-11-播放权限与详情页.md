# 第10+11步：播放权限控制与电影详情页

## 目标

实现后端播放权限校验（VIP影片限制）和前端电影详情页（含主创展示、评论系统、播放按钮、VIP充值引导）。

## 核心产出

| 文件 | 作用 |
|------|------|
| `service/PlayService.java` | 播放服务接口 |
| `service/impl/PlayServiceImpl.java` | 播放服务实现（权限校验 + 记录播放 + 历史查询） |
| `dto/PlayHistoryVO.java` | 播放记录返回VO |
| `controller/PlayController.java` | 播放接口控制器（播放/历史记录） |
| `controller/ReviewController.java` | 评论接口控制器（获取评论列表/提交评论） |
| `views/MovieDetail.vue` | 电影详情页（全宽海报+主创列表+评论+VIP引导） |
| `views/VideoPlayer.vue` | 观影页面 |
| `views/UserCenter.vue` | 个人中心（含播放历史） |

## API接口

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | `/api/movie/play/{movieId}` | 播放电影（需认证，返回播放URL） |
| GET | `/api/movie/history?page=1&size=10` | 用户播放历史 |
| GET | `/api/review/movie/{movieId}` | 获取电影评论列表（按时间倒序） |
| POST | `/api/review/submit` | 提交评论（需认证，支持同一用户多次评论） |

## 关键功能

- **播放权限校验**: `PlayServiceImpl.playMovie()` 先查电影状态，再检查 `is_vip` 字段。VIP影片校验用户角色是否为 VIP/ADMIN 或 `vip_expire_time` 未过期。非VIP访问VIP影片返回403。
- **播放记录**: **已移除同一用户同一天重复播放的唯一约束**，每次播放都新增一条记录并更新 `movie.play_count`。播放时自动清除周/月排行Redis缓存。
- **评论系统**: 用户可评分(1-10) + 文字评论。**已移除唯一约束 `uk_user_movie_review`**，同一用户可对同一电影发表多条评论（每条评论独立ID）。
- **MovieDetail.vue**: 顶部全宽海报+渐变覆盖层，左侧封面右侧信息。主创列表点击可跳转。评论区域已登录可提交评分+文字，未登录显示登录引导。VIP影片显示"VIP专享"金色按钮，点击弹出充值Dialog。
- **VideoPlayer.vue**: 支持视频播放（后端支持Range请求，拖拽进度）。
- **UserCenter.vue**: 展示用户的播放历史记录列表。

## 注意事项

- `/api/movie/play/{movieId}` 和 `/api/review/submit` 需要认证
- 评论表已 **无唯一约束**，同一用户可对同一电影多次评分评论
- 播放URL直接来自 `movie.play_url` 字段，视频文件存在本地 `tmdb_all_movie/` 目录
