# 第05步：用户注册/登录前端

## 目标

实现用户登录和注册的前端页面，包括表单校验、Token管理、路由守卫和演示账号快速登录功能。

## 核心产出

| 文件 | 作用 |
|------|------|
| `views/Login.vue` | 登录页面（含演示账号快速登录） |
| `views/Register.vue` | 注册页面（含确认密码校验） |
| `router/index.ts` | 路由配置 + 路由守卫（已建） |
| `stores/user.ts` | 用户状态管理（已建） |
| `api/index.ts` | Axios拦截器（已建） |

## 关键代码说明

- **Login.vue**:
  - 表单校验：用户名必填3-20字符，密码必填6-20字符
  - 登录成功跳转回来源页面（通过 `route.query.redirect`）
  - 底部提供3个演示账号快速登录按钮：admin(管理员)、vipuser(VIP)、normal(普通)
  - 深色渐变背景 + 毛玻璃效果卡片（`backdrop-filter: blur(20px)`）
  - 胶囊输入框 + 圆角按钮，符合电影主题风格

- **Register.vue**:
  - 字段：用户名、邮箱（必填）、手机号（选填+正则校验）、密码、确认密码
  - 自定义校验规则 `validateConfirmPassword` 确保两次密码一致
  - 注册成功后自动登录并跳转首页

## UI/UX要点

- 页面居中卡片布局，背景使用深蓝色渐变
- 输入框使用深色半透明背景，聚焦时显示红色边框（`--primary-color`）
- 登录按钮全宽 + 加载状态
- 演示账号区域可点击快速填充

## 注意事项

- 路由守卫在 `router/index.ts` 中：`/user`、`/payment` 等需要登录才能访问
- 401响应时 Axios 拦截器自动清除 Token 并跳转 `/login`
- 手机号为正则校验 `^1[3-9]\d{9}$`
